# Usage

Quick start for day-to-day SuperCollider work in Zed.

1) Configure the LSP launcher (see `docs/SETTINGS.md`).
2) Add SuperCollider tasks (see `docs/TASKS_SNIPPET.md`).
3) Open a `.sc`/`.scd` file.
4) Use the gutter play button on runnable blocks or a task keybinding to evaluate.
5) Use the Boot/Stop tasks to control the audio server.

Output appears in the terminal panel from the task/launcher process.
